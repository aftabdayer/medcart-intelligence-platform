# MedCart Intelligence Platform
### End-to-End Analytics for an Indian Online Pharmacy

**Author:** Aftab Dayer | **Stack:** Python · SQL · Power BI · Scikit-learn

## What I Built
- SQLite database with 7 tables, 12,000 orders, 600 patients, 55 drugs
- Python EDA generating 10 professional charts
- ML demand forecasting (Gradient Boosting, 17 features)
- 4-page Power BI dashboard

## Key Findings
- ₹4.62M total revenue across 3 years
- 37% of patients are chronic — generating 2x higher lifetime value
- 20% of inventory at risk (low stock or near expiry)
- Antibiotics + Diabetes = top revenue categories
